# 🛒 Carrinho de Compras (Simulação Shopee)

Projeto que simula um carrinho de compras inspirado na Shopee, desenvolvido com **JavaScript e Node.js**.

A aplicação permite adicionar, remover e listar produtos, além de calcular automaticamente o valor total da compra.

## 🚀 Funcionalidades

* Adicionar produtos ao carrinho
* Remover produtos
* Listar itens do carrinho
* Cálculo automático do total

## 🛠️ Tecnologias utilizadas

* JavaScript
* Node.js

## 🎯 Objetivo

Este projeto foi criado com o objetivo de praticar:

* Lógica de programação
* Modularização de código
* Manipulação de arrays e dados

Durante os estudos na **DIO**.
